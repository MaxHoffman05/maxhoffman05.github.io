Senior Capstone Project - TailTracker

For my Senior Capstone Project I worked in a group as a back-end coder and the project documentor. We worked in an Agile Team to complete the project on time with all the features our client wanted. We chose Javascript for the front-end and Python and Python Flask for the back-end. Our client needed a solution to helping her create Dog Agility Competition Schedules easier and faster. We were tasked with automating this process.

My responsibilities for the back-end were to create a way to parse through input excel files and calculate time alloted for each Dog Agility Class, in order to build out the schedule for the whole Dog Competition. Me and another teammate were also tasked with creating the API routes for the front end to access the back-end through Python Flask.

I was also tasked with documenting all project methodologies, project tools, and explaining my team's though processes into a documentation binder. I made many charts, images, edited screenshots, to illustrate how my team completed the project. This documentation binder had multiple parts, the most important being the User Manual, and the Technical Manual. The User Manual had to explain how to use our application to anyone, including someone who may not have much comptuer experience. The Technical Manual focused on explaining how our project worked to another development team and to give them all the tools they would need if someone else needed to maintain the project.

I believe this project helped me prepare for the real world and allowed me to strengthen and grow new skills like documentation and the Agile Method.
