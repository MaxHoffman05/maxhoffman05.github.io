  • > Cover Letter Wizard - Oct 2024

    • I developed this short Python script to semi-automate the process of creating cover letters. I knew that my previous method of creating cover letters could be improved and I could cut down on the work I knew could be automated. Examples of this include: automatically importing the date, changing the name of the company and position in my opening paragraph. Another reason I wanted to create this script was so I could store and display all the different cover letter paragraphs I have previously written to make them more easily accessible if they were applicable to a job I am currently applying for. I made this to be able to focus all my effort on creating the best cover letter instead of changing dates, company names etc.

    git clone https://github.com/MaxHoffman05/coverLetterWizard
